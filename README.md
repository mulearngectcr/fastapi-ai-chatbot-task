# Task F: AI Chatbot with FastAPI & Streamlit

## Overview

In this task, you will build and deploy a full stack AI chatbot. The backend is built using FastAPI and communicates with Groq's LLM API. The frontend is built using Streamlit (or a framework of your choice) and provides a chat interface.

---

## Submission Guidelines

### 1. Fork the Repository

Click the **Fork** button at the top-right of this repository.

### 2. Clone Your Fork

```bash
git clone https://github.com/<your-username>/fastapi-ai-chatbot.git
cd fastapi-ai-chatbot
```

### 3. Create a Folder with Your Name

Use lowercase letters and hyphens — no spaces.

```
fastapi-ai-chatbot/
└── john-doe/
    ├── backend.py
    ├── frontend.py
    ├── requirements.txt
    ├── .env.example
    ├── memory.json
    └── .gitignore
```

> ⚠️ Place all files inside your named folder. Do not put them in the root of the repo.

### 4. Commit and Push

```bash
git add .
git commit -m "Add submission - John Doe"
git push origin main
```

### 5. Open a Pull Request

1. Go to your fork on GitHub
2. Click **Contribute → Open Pull Request**
3. Set the PR title to: `Submission - John Doe`
4. Add both your deployed URLs in the PR description:

```
Backend URL: https://your-backend.onrender.com
Frontend URL: https://your-chatbot.streamlit.app
```

> 📌 Your submission is complete once the PR is opened.

---

Happy Building!
